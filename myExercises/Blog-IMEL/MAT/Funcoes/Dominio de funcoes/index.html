Tema: O Famoso Domínio de Funções




O domínio de uma funcao é o conjunto de valores que são permitidos para entrar na nossa função. Este conjunto são os valores de X para esta função


Podemos pensar na função como uma máquina que permite um certo número de igredientes e a cada igrediente que colocamos nela é gerado um produto para aquele igrediente



Como vímos no post anterior, nos falamos do tipos de funções algébricas , onde dissemos que elas podem ser racionais inteiras, fraccionárias e irracionais


Bom, hoje vamos falar de algo diferente, falaremos dos valores que são permitidos em cada função.


Então prepara as malas e vamos embarcar nesta viagem!!!


Analisemos a seguinte função :


y = 1 x

Como sabemos não podemos dividir por zero, então no nosso domínio não podemos incluir o valor "0". De qualquer modo se colocarmos qualquer outro número real, a função (a nossa máquina) vai funcionar muito bem.


Agora vamos definir o domínio de cada função que aprendemos pesssoal:


1.Funcao Algebrica Racional Inteira


Nesta tipo de função o domínio são todos os números reais. ya só mesmo isso


Como assim?


Bem vamos analizar a seguinte função:


y = 2 x + 1

Se colocamos qualquer valor nessa função, não teremos nenhum erro:


x f(x)
-1 -1
0 1
1 3
2 5
3 7
... ...


2.Funcao Algebrica Racional Fraccionaria


Nesta tipo de função o domínio são todos os números reais que não façam que o demoninador seja igual a zero. ya só mesmo isso


Como assim?


Bem vamos analizar a seguinte função:


y = 3 x - 1

Qual é o valor de X que poderar deixar o denominador igual à zero? Claro! o 1


Mas como fazemos isso, com equação niga:


1-Igualamos o demoninador à zero: x-1 = 0


2-Isolamos o x: X = 1


3- Colocamos o sinal de diferente X dif 1, ou seja x deve ser diferente de zero
